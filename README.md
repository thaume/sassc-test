# Sassc test

Install dependencies
```shell
npm install
```

Run grunt
```shell
grunt
```

To turn libsass on, uncomment this task :
```javascript
grunt.loadNpmTasks('grunt-sass');
```

 and comment the other one (this is the Ruby version of sass):
 ```javascript
grunt.loadNpmTasks('grunt-contrib-sass');
```
